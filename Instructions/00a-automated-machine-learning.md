## 指示
在實驗室中我們將看到怎樣在 Azure Machine Learning.中使用自動化機器學習。

1.	透過 https://docs.microsoft.com/learn/modules/use-automated-machine-learning/ 前往 Microsoft Learn 課程模組並在瀏覽器中完成下列單元： 

- 建立 Azure Machine Learning 工作區

- 探索資料 
- 訓練機器學習模型 
- 部署模型作為服務
