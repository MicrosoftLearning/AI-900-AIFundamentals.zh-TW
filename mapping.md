# <a name="ai-900-trainer-lab-mapping"></a>AI-900 訓練人員實驗室對應

AI-900 課程：「AI 基本概念」有一個實驗室子集指定用於導學型訓練，適合為期 1 天的課程採用。

AI-900 訓練人員實驗室是可在 Microsoft Learn 上找到的 AI-900 實驗室子集。

> 注意：教學時，講師可選擇從所有 AI-900 實驗室集合中，新增額外的實驗室到他們的課程模組。
