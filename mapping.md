# AI-900 訓練人員實驗室對應

本課程 AI-900：AI 基本概念，有一部分實驗室，專為適合 1 天課程而調整的講師導向訓練所指定。

AI-900 訓練人員實驗室是可在 Microsoft Learn 上找到的 AI-900 實驗室子集。

> 注意：講師可以選擇在教學時，從完整的 AI-900 實驗室集新增其他實驗室到其課程模組。
